# Words Of The Web

[Website](https://wordsoftheweb.web.app)

## Deploy

`firebase deploy`

## Dev

To run the emulators: `firebase emulators:start`

Enabling Chrome Debugging

In shell:

`/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome --remote-debugging-port=9222`
